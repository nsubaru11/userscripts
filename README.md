# userscripts

個人的に作成した UserScript のまとめです。

## スクリプト一覧

### Gemini Full-Width Interface
Gemini の UI を最適化し、チャット画面の幅を広げます。また、ユーザーの吹き出しを右側に寄せるなどのレイアウト調整を行います。

- **[インストール・ダウンロード](https://raw.githubusercontent.com/nsubaru11/userscripts/main/src/gemini-full-width-interface.user.js)**

### Moodle Wakayama Univ Auto-Login
和歌山大学の Moodle および Microsoft 365 ログインページで自動ログインを試みます。

- **[インストール・ダウンロード](https://raw.githubusercontent.com/nsubaru11/userscripts/main/src/moodle-autologin.user.js)**

### Paiza Language Selector
paiza のコード提出ページで、あらかじめ設定したプログラミング言語に自動で切り替えます。

- **[インストール・ダウンロード](https://raw.githubusercontent.com/nsubaru11/userscripts/main/src/paiza-language-selector.user.js)**

## 使い方
1. Tampermonkey などのブラウザ拡張機能をインストールします。
2. 上記のリンクをクリックしてスクリプトをインストールしてください。
